# ASIX 2n · Itinerari Ciberseguretat

Benvinguts a la documentació del projecte **NetCorp S.L.**: l'empresa fictícia
sobre la qual treballarem durant tot el curs amb una metodologia d'**Aprenentatge
Basat en Projectes (ABP)**.

## Com funciona aquest curs

Al llarg del curs desenvoluparem **6 projectes transversals**, cadascun ampliant
la infraestructura de NetCorp S.L. construïda en els projectes anteriors. Cada
projecte combina continguts de diversos mòduls (M06, M07, M08, C037, C038) que
es treballen de manera coordinada.

## Els projectes

| Projecte | Títol | Mòduls implicats |
|---|---|---|
| **P1** | Posem en marxa la xarxa corporativa | M08, M06, M07, C037 |
| **P2** | Identitat corporativa: directori, VLANs i DNS | M08, M06, M07, C037 |
| **P3** | Publiquem serveis: web, FTP, correu i accés remot | M08, M07, M06, C037 |
| **P4** | Col·laboració, mobilitat i primera auditoria | M07, M06, C037, C038 |
| **P5** | Blindem el perímetre: tallafocs, VPN i monitoratge | C037, C038, M06, M08 |
| **P6** | Incident! Resposta, forense i resiliència | C038, C037, M06, M07 |

Cada projecte té la seva pròpia **web interactiva** (pestanyes amb resum, teoria,
recursos, pràctica detallada per fases i avaluació), accessible des del menú
**Projectes**.

## Material teòric

A l'apartat **Teoria** trobaràs els documents de fonaments necessaris per a
cada mòdul, pensats per estudiar de manera autònoma i com a suport a les
explicacions de classe.

!!! tip "Com estudiar"
    Per a cada projecte, primer llegeix la pestanya **Teoria** de la seva web
    i/o el document de teoria corresponent, i després segueix la pestanya
    **Pràctica** pas a pas.
