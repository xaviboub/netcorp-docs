# Visió general dels projectes

Aquest curs es desenvolupa al voltant de **NetCorp S.L.**, una empresa fictícia
de distribució logística amb seu central a Barcelona i delegacions a Girona i
Tarragona (aquesta última allotjada al cloud, a Azure).

Cada projecte construeix sobre la infraestructura dels anteriors. Fes clic a
cada projecte al menú lateral per accedir a la seva web interactiva completa
(resum, teoria, recursos, pràctica i avaluació).

## Fil conductor

```
P1 → Xarxa bàsica (switch, router, DHCP, VPN a Azure)
P2 → Identitat (VLANs per departament, Active Directory, DNS intern)
P3 → Serveis (web, FTP, correu, NAT, encaminament dinàmic)
P4 → Col·laboració i mobilitat (xat, videoconferència, impressió, IoT) + 1a auditoria
P5 → Perímetre (DMZ, tallafocs, VPN per a teletreball, IDS/IPS) + re-test
P6 → Incident simulat: resposta, anàlisi forense i recuperació
```
